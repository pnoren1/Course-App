# Product Overview

Course Portal is a learning management system (LMS) built for educational institutions. It provides a comprehensive platform for managing courses, users, assignments, and student progress.

## Core Features

- **Multi-method Authentication**: Google OAuth and username/password authentication
- **User Management**: Admin panel for creating users, sending invitations, and managing roles
- **Role-Based Access**: Support for student, instructor, moderator, and admin roles
- **Organization Structure**: Users can be grouped into organizations and sub-groups
- **Assignment System**: Create, distribute, and manage assignments with file submissions
- **Progress Tracking**: Monitor student progress across units and courses
- **Video Content**: Track video viewing progress and completion
- **Email Broadcasting**: Send targeted emails to users and groups
- **Bulk Operations**: Import/export users via CSV, bulk user creation

## User Flows

**Admin User Creation** (two methods):
1. Direct Creation: Admin creates user with credentials → User can login immediately
2. Invitation Flow: Admin sends invitation → User accepts via link → Profile created

**Student Experience**:
- Login → View course content → Watch videos → Submit assignments → Track progress

## Primary Users

- **Students**: Access course materials, submit assignments, track progress
- **Instructors**: Manage course content, review submissions, grade work
- **Admins**: Full system access, user management, organization setup, reporting

## Language

The system is primarily in Hebrew (RTL interface) with bilingual support in documentation.
