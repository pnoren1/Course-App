# תיעוד מערכת ניהול הקורסים

ברוכים הבאים לתיעוד המלא של מערכת ניהול הקורסים. התיעוד מחולק לשלושה תחומים עיקריים:

## 📋 מדריכי התקנה (Setup Guides)

מדריכים להתקנה והגדרה ראשונית של רכיבי המערכת:

- **[Email Setup](./setup/email-setup.md)** - הגדרת מערכת המייל לשליחת הודעות למשתמשים
- **[Storage Setup](./setup/storage-setup.md)** - הגדרת אחסון קבצים ב-Supabase Storage
- **[Assignment Management](./setup/assignment-setup.md)** - הגדרת מערכת המטלות וההגשות

## 📖 מדריכי שימוש (User Guides)

מדריכים מפורטים לשימוש בתכונות המערכת:

- **[Authentication Guide](./guides/authentication.md)** - מערכת ההתחברות עם שם משתמש וסיסמה
- **[User Roles & Organizations](./guides/user-roles.md)** - ניהול תפקידים וארגונים
- **[Bulk User Import](./guides/bulk-import.md)** - ייבוא וייצוא משתמשים באמצעות CSV
- **[Email Broadcasting](./guides/email-broadcast.md)** - שליחת מיילים קבוצתיים למשתמשים

## 🔒 אבטחה (Security)

תיעוד שיפורי אבטחה ונהלים:

- **[Security Improvements](./security/security-improvements.md)** - תיעוד מלא של שיפורי האבטחה במערכת

## תרומה לתיעוד

כשמוסיפים תכונות חדשות למערכת:
1. הוסף מדריך מתאים לתיקייה הרלוונטית
2. עדכן את README.md הראשי בשורש הפרויקט
3. עדכן את קובץ זה עם קישור למדריך החדש

## שאלות ותמיכה

לשאלות או בעיות, פנה למנהל המערכת או פתח issue בפרויקט.
